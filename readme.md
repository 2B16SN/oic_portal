## Get Started

#### Clone
```
git clone https://github.com/miniikit/oic_portal
```

#### Install
```
composer install
```

#### .env Copy
```
cp .env.example .env
```

#### Make Key
```
php artisan key:generate
```

#### Migration
```
php artisan migrate:refresh
```

#### DB Seed
```
php artisan db::seed
```

#### Package Install
```
npm install
```

#### Build
```
npm run dev
```

#### Up Server
```
php artisan serve
```
